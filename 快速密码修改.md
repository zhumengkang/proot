# 🚀 LibreTV 密码快速修改指南

## 只需3步，30秒完成！

### 📍 步骤1：找到文件
```
文件位置: config/master-config.js
```

### ✏️ 步骤2：修改密码
找到第8行：
```javascript
password: '123qwe!@#QWE',    // 🔥 改成您的密码
```

改为：
```javascript
password: '您的新密码',      // 🔥 您的新密码
```

### 💾 步骤3：保存文件
按 `Ctrl + S` (Windows) 或 `Cmd + S` (Mac)
